[NVIDIA](https://www.nvidia.cn/)是一家GPU显卡到高性能并行计算的人工智能计算公司，创立于1993年。

NVIDIA® Jetson™ 系统所提供的性能和能效可提高自主机器软件的运行速度，而且功耗更低。

每个系统都是一个完备的模块化系统 (SOM)，具备 CPU、GPU、PMIC、DRAM 和闪存。Jetson 还具备可扩展性。
